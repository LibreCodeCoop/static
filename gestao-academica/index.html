Gestão acadêmica

Um software para gestão acadêmica ou gestão escolar é um sistema para gerenciar a vida acadêmica da instituição como as tarefas de secretaria, inscrição em disciplinas, lançamento de notas entre outros.

Como exemplo de sistema para gestão acadêmica, temos o i-Educar que é um software livre e público, hospedado no Portal do Software Público Brasileiro. Foi desenvolvido para fazer gestão eficaz do sistema de ensino escolar, principalmente na esfera pública. O software já utilizado por algumas prefeituras brasileiras, como é o caso de Itajaí (SC), Arapiraca (AL), Porto Velho (RO) e Parauapebas (PA). Dentre seus recursos, temos:

Gestão de alunos

Cadastro e edição de dados dos alunos

Componentes curriculares

Cadastro e gerenciamento de cursos, disciplinas e áreas de conhecimento que serão lecionadas na rede de ensino.

Cadastro de servidores

Cadastro e gerenciamento de dados de servidores da rede se ensino. Gerenciamento de alocações, afastamentos, substituições, faltas e atrasos.

Quadro de horários

No cadastro de Quadro de horários é possível efetuar a gestão dos dias da semana, horários, disciplinas e professores para cada turma.

Integração com Educacenso/INEP

Importação e exportação automatizadas dos dados Censo Escolar em poucos cliques.

Sistema de avaliação flexível

Avaliação e relatórios adaptados com notas numéricas, conceituais ou avaliação descritiva.

Biblioteca

Gestão completa de acervo com controle de reservas, empréstimos e devoluções.

Transporte escolar

Controle de veículos, motoristas, rotas, itinerários e emissão de carteirinha.
